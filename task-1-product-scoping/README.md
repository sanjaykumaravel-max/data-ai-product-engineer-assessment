## What This Covers

## Key Decisions
- Prioritized internal analysts as the primary v1 user.
- Chose deterministic KPI-based guidance over predictive modeling for clarity and trust.
- Designed around existing tools/workflows instead of requiring process changes.
- Kept outputs exportable and consistent to reduce manual reporting variability.

## What I'd Revisit With More Time
- Add role-based views for managers and client-facing teams.
- Introduce anomaly detection and proactive alerting.
- Add scenario simulation for budget reallocation decisions.
- Explore advanced attribution/forecasting once data definitions are fully stabilized.
